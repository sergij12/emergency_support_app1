EmergencySupportApp
===================

Що це:
- готовий MVP-проєкт для Android Studio
- Kotlin + Jetpack Compose + Room + DataStore
- персональні плани дій, ресурси, чек-лист, довідник, режим тривоги

Що реалізовано:
- Onboarding
- Home
- Resources
- Planner
- Alarm mode
- Guide
- Checklist
- Local storage через Room
